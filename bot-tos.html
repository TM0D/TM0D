Luma Privacy Policy

Luma only passively records data required for its operation, such as user IDs, username, message IDs, and server IDs within it's database for use with it's various modules. This data is not shared with any third parties, and is only used for the operation of the bot. This bot does has the ability to access message contents and server members. This is only used for moderation, and is not stored or logged within the Database. This data is stored indefinitely, but can be deleted upon request. This data is not used for any other purposes, and is not used to track users outside of the bot's operation.

A breakdown of identifiable & non-passively recorded data, by database table, is stored as follows:

  • User ID (Levels, Birthdays, BoopBot, Starboard, GrazCoin)
  • Guild ID (Levels, Birthdays, BoopBot, Starboard)
  • Username (Levels, BoopBot)
  • XP (Levels)
  • Birthday (Birthdays)
  • Command Executions (BoopBot)
  • Message ID (Starboard)
  • Author ID (Starboard)
  • Channel ID (Starboard)
  • Pinner ID (Starboard)

The module that collects the most data is the Starboard module, which collects, message ID, author ID, channel ID, and pinner ID. This data is only used for the operation of the Starboard module, and is not used for any other purposes. This data is stored indefinitely, but can be deleted upon request. This data is not used for any other purposes, and is not used to track users outside of the bot's operation.

In addition to identifiable data stored within the database, the bot also records statistics on its operation, such as the number of messages processed, the number of commands executed, and the number of events received. This data is visible through the bot's /status command. This data does not have any link to and users. This data is stored indefinitely. This data is not used for any other purposes, and is not used to track users outside of the bot's operation.
